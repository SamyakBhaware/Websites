Quiz Instructions

Welcome to the quiz! Please read the instructions below carefully before starting the quiz.

  1. Number of Questions: The quiz consists of 10 multiple-choice questions.
  2. Answering: Select one option for each question. There is only one correct answer per question.
  3. No Negative Marking: There is no negative marking for incorrect answers, so feel free to attempt all questions.
  4. Completion: Once you finish the quiz, click the "Submit" button to see your results.
  5. Navigation: You can skip questions and return to them later. Use the "Next" and "Previous" buttons to navigate between questions.

Good luck! We hope you enjoy the quiz.
